Requirements:
- Colorama - Provides for text colorization.

During run, the following keys can be pressed to quit or pause the script
- q, Ctrl-C - Provides the user the opportunity to stop processing processing of the current file is complete
- p - Pauses the script after processing of the current file is complete, and waits for the user to initiate resumption

Important: There will be no immediate feedback once the above keys are pressed. You will not receive any feedback from the script until the processing of the current file are complete.
